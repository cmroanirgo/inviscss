InvisCss

Introduction

InvisCSS is designed to bring you the core of a good Css library without the cruft. That is, it's not filled with styles that are only used by developers and designers for their own websites. The default styling is minimal so that it's easier to extend and modify, by you. Unlike bootstrap and most CSS templates, which requires a developer to excessively 'opt-in' by specifying classes all over the place, InvisCSS uses a few simple 'opt-in' classes at the base of an HTML branch. InvisCSS is for real sites:

Cleaner HTML

  • No more class="classA classB classC" nonsense at each and every level of your HTML. InvisCSS makes your HTML is clean and lean again, using widely used naming conventions. Although built with a Bootstrap audience in mind, you'll love the simplicity.

Smarter CSS

  • Use of rem units makes font scaling a snap.
  • No coloring or assumptions made (except for buttons).
  • Forms are unbelievably easy.
  • Minimal rules means it's easier to override. InvisCss is designed to be easy to change core features.
